Bird Tweeter

An IoT bird feeder that tweets photos of visiting birds using ESP32-CAM and Python.

About the Project

Bird Tweeter is an IoT bird feeder that captures photos of birds and posts them to Twitter. It uses an ESP32-CAM for wireless image capture, motion detection, and Python scripts for automation and species identification.

  • ESP32-CAM for wireless image capture and motion detection
  • Automated tweets with Python and Tweepy
  • Bird species identification using iNaturalist API
  • Filters out non-bird visitors (like squirrels)
  • Real-time notifications and logging
