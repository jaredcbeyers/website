Skull King

A digital version of the classic card game, built with C++ and a custom UI.

About the Project

Skull King is a digital adaptation of the popular trick-taking card game. This project was built in C++ and features a custom user interface, score tracking, and full game logic for solo or multiplayer play.

  • Turn-based gameplay with bidding and trick-taking mechanics
  • Custom UI for card selection and score display
  • Rule enforcement and win condition logic
  • Designed for both solo and multiplayer experiences
