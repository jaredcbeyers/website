RFID Project

A secure RFID-based access control system using Arduino and MFRC522.

About the Project

This project implements a secure access control system using RFID technology. Built with Arduino and the MFRC522 module, it manages user authentication, logs access events, and provides real-time feedback with LEDs and a buzzer.

  • RFID card/tag authentication for secure access
  • EEPROM storage for user management
  • Visual and audio feedback with LEDs and buzzer
  • LCD display for status and instructions
  • Real-time access logging
